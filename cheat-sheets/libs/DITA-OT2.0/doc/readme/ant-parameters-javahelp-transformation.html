Ant parameters: JavaHelp transformation

Certain parameters are specific to the JavaHelp transformation.

Figure 1. Ant parameters: JavaHelp transformation
args.javahelp.map
Specifies the name of the ditamap file for a JavaHelp project.
args.javahelp.toc
Specifies the name of the file containing the TOC in your JavaHelp output. Default value is the name of the ditamap file for your project.
