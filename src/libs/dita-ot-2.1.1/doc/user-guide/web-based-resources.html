Web-based resources

There are many vital DITA resources online, including the Yahoo! dita-users group and the DITA-OT project page at dita.xml.org.

DITA-OT project page at dita.xml.org
The DITA-OT project page at dita.xml.org provides news about the latest toolkit builds, plans for the next milestone release, and other rapidly-changing information. It also contains release notes for all past and upcoming releases.
